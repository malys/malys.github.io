Tests avec Internet Explorer

a year ago NodeJS, Karma, JasmineJS, ModernIE, Windows Malys

Objectif

Nous avons actuellement une batterie de tests se lançant grâce à Karma sur différents navigateurs. Nous souhaiterions pouvoir valdier nos tests sur l’ensemble des versions d’Internet Explorer que nous supportons.

Prérequis

Microfost fournit des machines virtuelles contenant différentes versions de systême d’exploitation avec leur version d’Internet Explorer associée.

Vous les trouverez sur ce site: ModernIE Pour automatiser la manipulation des vms,nous allons utiliser les outils suivant:

IEVMS pour l’installation des vms

IECTRL pour la gestion des vms

Karma-IEVMS pour utiliser Karma et les ModernIE

L’ensemble des outils a été développé par xdissent pour des Unix like OS.

Il va falloir faire quelques manipulations pour les faire fonctionner sous Windows.Il faut avoir Cygwin installé sur son poste et présent dans la %PATH% ainsi que VirtualBox.

Pour ajouter la compatiblité avec Windows, j’ai créé un fork avec le pull request associé.

Il faut appliquer manuellement les correctifs sur ievms.js si le pull request n’a pas été intégré. Cette modification impacte le project IECTRL et Karma-IEVMS.

Planification

Une tâche planifiée va supprimer et recréer les vms.

cmd /C uninstall.cmd
cmd /C launch.cmd
exit
Rem uninstall.cmd
Rem Remove VMS
@ECHO OFF
SET LIST="IE8 - WinXP","IE9 - Win7","IE10 - Win7"

SET LIST=ECHO %LIST:,=^^^&ECHO %

FOR /F "delims=" %%i IN ('%LIST%') DO (
	echo iectrl stop %%i
	cmd /C "iectrl stop %%i"
	echo iectrl uninstall %%i
	cmd /C "iectrl uninstall %%i"
)
exit
